Run a Model

To run new models first select the Market Area you wish to run a model for then click Run New Model nav. From here you can modify the model parameters and start a model run.

Model Analysis

To view and analyze model output first select the market area you wish to analyze. This will load the list of completed modeling scenarios for that market area. Select the Active Model then navigate to Overview, Route, Stop or Map analysis.
